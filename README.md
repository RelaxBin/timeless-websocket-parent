# timeless-websocket-parent
2023/6/27：
项目初始化，vue + springboot + websocket写的聊天小案例。
主要功能：
1. 注册和登录
2. 私人聊天
3. 保存聊天记录
4. 清除聊天记录
5. 显示在线人数
